# compose-timetable
컴포즈로 동작하는 대학교 시간표 with KOIN (한국기술교육대학교)

## Skill
- Jetpack Compose
- Hilt
- Gson
- SharedPreference DataStore
- Media Image
- Jetpack Compose Glance

## Asset
- timetable.json (강의)
- department.json (학과)
- semester.json (학기)
